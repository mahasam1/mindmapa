Controls:

  • Double-click: Create a new node.
  • Drag node: Move a node.
  • Click node: Select a node.
  • Type: Edit selected node's text.
  • Right-click & drag: Connect nodes.
  • Mouse wheel: Zoom in/out.
  • Middle-click & drag: Pan the canvas.
  • ESC: Clear all nodes and reset.
  • Ctrl+V (on selected node): Attach URL from clipboard.
  • Ctrl + Click (on node with URL): Open URL in new tab.
  • Save Map button: Save current mind map to a .dimap file.
  • Load Map button: Load mind map from a .dimap file.
