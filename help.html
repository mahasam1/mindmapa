Mind Mapper Controls

Node Management

  • Double-click: Create a new text object (text only, no shape).
  • Drag & Drop Node: Move a node. Drop on another node to make it a child.
  • Click Node: Select a node or text object.
  • Type: Edit selected node's or text object's text.
  • Tab (on selected node): Create a new child node (not available for text objects).
  • Enter (on selected node): Create a new sibling node.
  • DEL: Delete selected node and its children.
  • \: Toggle fold/unfold children of selected node.

Canvas Control

  • Mouse Wheel: Zoom in/out.
  • Middle-click & Drag: Pan the canvas.
  • Right-click & Drag: Connect two nodes.
  • ESC: Clear all nodes and reset the map.

Node & Text Styling

  • + (on nodes): Increase size of selected node.
  • - (on nodes): Decrease size of selected node.
  • + (on text objects): Increase font size of selected text object.
  • - (on text objects): Decrease font size of selected text object.
  • Ctrl+Shift: Open color picker for selected node and its children.

Attachments

  • Ctrl+V (on selected node): Paste URL, Image, or Text from clipboard.
  • Ctrl+V (on text objects): Paste URL or Image (text pasting creates child nodes only for regular nodes).
  • Ctrl + Click (on node/text object with URL): Open URL in new tab.
  • Ctrl+DEL: Delete image from selected node or text object.
  • Ctrl + '+': Increase size of image on selected node or text object.
  • Ctrl + '-': Decrease size of image on selected node or text object.

Text Objects

  • What are text objects: Shape-free text elements created with double-click.
  • Key differences: No visual shape, only text with green glow when selected.
  • Size control: +/- keys change actual font size (8px to 72px).
  • Cannot have children: Tab key and text pasting as children are disabled.
  • Can be reparented: Can still be moved and made children of regular nodes.
  • Full feature support: URLs, images, colors, undo/redo all work normally.

History

  • Ctrl+Z: Undo last action.
  • Ctrl+Y: Redo last action.

Map Management

  • Save Map button: Save current mind map to a .dimap file.
  • Load Map button: Load mind map from a .dimap file.
